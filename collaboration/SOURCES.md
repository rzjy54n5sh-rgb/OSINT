# MENA Intel Desk — Source Registry

> Agreed source classifications. Both analysts use this as the common epistemological baseline.  
> If you disagree with a classification, open a DEBATES.md entry — don't silently deviate.

---

## Tier 1 — Primary Sources (Highest Weight)

These can be cited directly in reports without hedging.

| Source | Type | Notes |
|---|---|---|
| Reuters | Wire service | Global standard, multi-bureau verification |
| AP (Associated Press) | Wire service | Strong MENA bureau coverage |
| AFP | Wire service | Strong Francophone MENA coverage |
| BBC World Service | Broadcast/digital | Strong editorial standards, Arabic service available |
| Al Jazeera English | Digital/broadcast | Qatar-funded — acknowledge Gulf perspective, strong on-ground access |
| ISW (understandingwar.org) | Think tank | Daily Ukraine + Middle East assessments. Pro-Western framing — note when relevant |
| ACLED (acleddata.com) | Data institution | Academic-grade conflict event database, downloadable |
| Crisis Group (crisisgroup.org) | Think tank | Strong on conflict resolution framing |
| Bellingcat | Investigative | OSINT verification standard. Pro-Western lean — note when relevant |

## Tier 3 — Monitor Only (Never Cite Directly)

Track for signal, but independently verify before including in any report.

| Source | Why Tier 3 |
|---|---|
| IRNA (Iran state agency) | State propaganda arm — useful only for official Iranian positions |
| PressTV | Iranian state broadcast — same as IRNA |
| Al Mayadeen | Hezbollah-aligned — useful for proxy perspective only |
| Telegram channels (war footage) | Unverified, high noise, high disinformation risk |
| X/Twitter (unverified accounts) | Useful for real-time signal detection only |

| Times of Israel opinion | Opinion, not news — label clearly |

## Known Bias Profiles

| Source | Known Lean | Mitigation |
|---|---|---|
| Al Jazeera | Qatar foreign policy alignment, pro-Islamist on some issues | Cross-check against Reuters/AP on same story |
| ISW | Pro-Ukraine, pro-Western intervention | Noted on all ISW citations in reports |
| Bellingcat | Pro-Western OSINT — tends to investigate Russia/Iran more than Western actors | Acknowledge scope bias |
| Middle East Eye | Pro-Muslim Brotherhood, pro-Palestinian | Useful for those perspectives, not as neutral baseline |
| Israeli mainstream media | Domestic audience framing | Useful for Israeli political dynamics, not for regional balance |

---

## Red Flag Patterns — Automatic Downgrade to Tier 3

Any source exhibiting these patterns is immediately downgraded regardless of normal tier:
- Recycled footage from prior conflicts misattributed to current events
- AI-generated satellite imagery (check metadata and resolution artifacts)
- Unverified casualty figures without hospital/NGO confirmation
- Claims of Hormuz "closure" without tanker diversion data to support
- Price projections presented as forecasts without model attribution
